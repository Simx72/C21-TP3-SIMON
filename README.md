## Compilation

Sur linux...

Install necessary components

```
sudo apt install cmake ninja-build
```

Then

```bash
# Create build instructions
cmake --preset linux
# Compile
cmake --build --preset linux
```

```bash
# Run
build/linux/TP3
```
